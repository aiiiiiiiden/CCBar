This directory contains Architecture Decision Records (ADRs) for the CCBar project.

## Index

| ID | Title | Status |
|----|-------|--------|
| [ADR-0000](0000-template.md) | ADR Template | - |
| [ADR-0001](0001-swiftnio-for-http-server.md) | SwiftNIO for HTTP Server | Accepted |

## What is an ADR?

An ADR captures a significant architectural decision along with its context and consequences. Each ADR is a short text file describing a single decision.

## Creating a New ADR

1. Copy `0000-template.md` to a new file with the next sequential number
2. Fill in each section
3. Set the status to `Proposed`
4. Add an entry to the index table above
5. Submit a PR for team review
